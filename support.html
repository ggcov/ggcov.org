--- title: ggcov Support ---

Support

If you find a problem with ggcov, you can:

  • Report a bug, or
  • Ask on the mailing list

Contributing

If you fix a bug or port ggcov to a new platform, please post your changes so they can be incorporated into the next release. A pull request on GitHub is the best choice, but you can also post a unified format patch to the mailing list.
