Got code?

Sure you do. Lots and lots of code. And tests...not so much? Not enough?

So you know you need more tests, because everyone needs more tests... but how are you supposed to know which parts of your code to test next?

Easy: ggcov tells you which parts of your software are well tested, and which parts...need a little lovin'. ggcov is a set of tools for exploring test coverage data produced by C and C++ programs compiled with gcc --coverage. So it's basically a replacement for the gcov program that comes with gcc.

ggcov is released under the GNU General Public License version 2 (GPLv2) with no warranty.

ggcov has a tool that's convenient for your workflow.

Got Desktop?

ggcov has a Gnome GUI for exploring test coverage with visualisations and point and click.
